저작권

이 사이트는 기상청 API 허브에서 제공하는 API를 이용한 사이트입니다.

본 저작물은 기상청에서 23년 작성하여 공공누리 제1유형으로 개방한 '기상청 API허브'를 사용하였으며, 해당 저작물은 기상청 API허브(apihub.kma.go.kr)에서 무료로 다운받으실 수 있습니다.
